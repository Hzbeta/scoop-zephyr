## 如何从此 bucket 安装应用？

在 PowerShell 中运行以下命令来添加 bucket：

```pwsh
scoop bucket add zephyr https://github.com/Hzbeta/scoop-zephyr
```

使用以下命令从此 bucket 安装应用：

```pwsh
scoop install zephyr/<应用名称>
```
